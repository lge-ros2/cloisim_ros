# CLOiSim-ROS Protobuf Interfaces

Defines protobuf message formats for communication between CLOiSim (Unity multi-robot simulator) and ROS nodes.

## Overview

This package contains 231 protobuf message definitions, of which 21 are actively used for sensor data transport and control commands. Messages are organized by functionality including sensor data, control commands, and world information.

## Message Directory

All proto files are located in the [msgs/](msgs/) directory.
